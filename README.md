# Mission Labo - cadeau d'anniversaire

Mini-jeu HTML/CSS/JavaScript prêt pour GitHub Pages.

## Comment l'utiliser

1. Remplace les fichiers dans `assets/` :
   - `musique.mp3` par ta chanson
   - `renne.png` par la photo finale de Louise en renne
2. Mets tout le dossier sur GitHub.
3. Active GitHub Pages :
   - Settings
   - Pages
   - Deploy from a branch
   - Branch: `main`
   - Folder: `/root`
4. Le jeu sera disponible via l'URL GitHub Pages.

## Contrôles

- Ordinateur : flèches du clavier, ou ZQSD/WASD.
- Mobile : boutons tactiles en bas de l'écran.

## But du jeu

Louise doit récupérer :
- 🔬 microscope
- 🧫 culture bactérienne
- ☕ café

Ensuite, elle doit rejoindre 🎁 sans se faire attraper par Pascal.

Si Pascal l'attrape :

> Pascal apparaît.  
> “Tu as une minute pour discuter des résultats ?”  
> Retour au début du labo.
